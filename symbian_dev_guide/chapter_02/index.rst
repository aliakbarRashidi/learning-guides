Using Qt Creator to Setup the Project
=====================================

In order to follow the steps of this guide, you'll first need to set up your environment, understand the main components of a basic Symbian^3 application and be able to deploy it onto a device.

For this purpose, this chapter illustrates how to build a     Hello World* Symbian^3 application using :creator:`Qt Creator IDE <index.html>` on Windows. Please note that the screenshots presented in this guide are generated using Qt Creator 2.3 and may be different on your version.
